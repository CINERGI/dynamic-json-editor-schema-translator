Mouse

  • End Point Nodes- Click to edit
  • Parent Nodes - Click to collapse or expand

Metadata Tree Display Legend

Links and Nodes
  • End Point Node - is the last element in a tree. These can be edited. If the node has a value, the value will be displayed, else it will show the name of the field
  • Parent Nodes - Objects and Arrays - Objects are Blue circles with children. Arrays are Red circles with children. They will always display the field name above the node.
  • Links - Links that have populated data will have a Gold color, empty links will have grey links.
  • Node Highlites - If a node contains a required dataset it will either be highlighted in green (it has been populated) or red (its empty and required editing).

Loading Datasets

NGDS CKAN data source:
  • Enter Search term and click the spyglass search icon to bring a list of datasets
  • Next & Back - Moves to the next or previous page of datasets
  • Page Size - you can change the page size that determines the number of datasets that are displayed.
  • Page Number - you can change the page directly and click next to move to that page
  • Pull a list - select next while the search term is empty to list all datasets

Editing Metadata

Data Types that can be edited:
  • Text - Edit, delete or add text
  • Large Text Areas - resizable object to handle large text area.
  • Arrays - rigth click on an array (the Red Circles ) and you can add new elements or delete elements to the array
  • Parent Nodes - rigth click on an array (the Blue Circles ) and you can add new elements or delete element elements to the object. You can only add objects if they do not already exist.
  • Map Bounding Box -
  • Dates - Pick a date from calendar
  • GUIDS - Guids can only be created and not edited. Click on an empty guid to create a new on

Metadata Schemas

Answer:

JSON schemas translate the incoming data to the tree structure displayed, and transform it back when saved. To change the tree view and the data to be edited, requires a new json schema to be uploaded.

JSON Data Dictionaries - populate pick lists for select fields.
